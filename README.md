# Portfolio 2026

Single-page DevOps & Systems Engineer portfolio built with Next.js and Tailwind CSS. The layout is designed to feel enterprise-grade with a clean, Apple-like polish and a modern terminal-inspired hero.

## Features
- Single-page layout with section navigation
- Terminal-style hero card
- Scroll-to-top control and section rail
- Tailwind CSS theming with blue/indigo gradients

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Getting Started

### 1) Install dependencies
```bash
npm install
```

### 2) Start the development server
```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

### 3) Build for production
```bash
npm run build
```

### 4) Run the production server
```bash
npm run start
```
